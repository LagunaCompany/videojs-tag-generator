Feedback and Errors

If anything goes wrong, you'll see useful messages below.

Errors will definitely need your attention. Warnings should be looked at; things will probably still work, but maybe not as you expect.

Error: Enter at Least One Video URL

Without any video URLs, this page can't create a usable video tag for you!

Warning: Preload and autoplay will not work on iOS devices

For technical information, please see Apple's official documentation.

Warning: Autoplaying Video, and the Custom Play Button

The video has been set to autoplay, so the custom play button won't be used!

Warning: Looping Video, and the Custom Replay Button

The video has been set to loop, so the custom replay button won't be used!

Warning: Looping Video, and End Screen Content

The video has been set to loop, so the end screen content won't be used!

Warning: End Screen Content, and the Custom Replay Button

Because you have chosen to use a custom replay button and end screen content, the layout may be a little messy as the player tries to show both when the video ends.
